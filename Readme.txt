To create some changes you can use 2 instruments:

A. simple and primitive method:
  1) do some changes in file
  2) add changes to index use $ git add Readme.txt
  3) create commit use $ git commit -m "you text"

B. use a branches
  1) create new branch use $ git branch branchname
  2) switch to this branch use $ git checkout branchname
ITS INTERESTING! you can substitute these commands with $ git checkout -b branchname
  3) do dome changes
  4) add changes to index use $ git add Readme.txt
  5) create commit use $ git commit -m "you text"
  6) switch to master branch use $ git checkout master
  7) do merge use $ git merge branchname
  8) delete branch use $ git branch -d branchname
